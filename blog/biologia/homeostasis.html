Homeostasis

Los seres vivos no están en equilibrio pasivo con el ambiente. Mantienen condiciones internas dentro de rangos adecuados. A eso se le llama homeostasis.

Idea claveEstabilidad no significa ausencia de cambio.

El cuerpo ajusta variables constantemente para mantenerse dentro de rangos funcionales.

Qué regula un organismo

Un organismo puede regular temperatura, agua, glucosa, pH y concentración de sales, entre muchas otras variables.

Si esas variables se alejan demasiado de ciertos rangos, las células dejan de funcionar bien.

Ejemplo: temperatura

Cuando tienes calor, sudas. Al evaporarse el sudor, ayuda a perder energía térmica. Cuando tienes frío, puedes temblar para generar calor.

Ambas respuestas buscan mantener la temperatura interna dentro de un rango.

Retroalimentación negativa

Muchos sistemas homeostáticos funcionan con retroalimentación negativa: si una variable sube demasiado, se activan respuestas para bajarla; si baja demasiado, se activan respuestas para subirla.

Mini ejercicio

Si aumenta la glucosa en sangre después de comer, el cuerpo activa mecanismos para reducirla. Esa corrección es un ejemplo de regulación homeostática.

Por qué importa

La homeostasis permite que las células trabajen en condiciones relativamente estables aunque el exterior cambie. Sin regulación, pequeñas variaciones podrían afectar enzimas, membranas y reacciones químicas.

Por eso fiebre, deshidratación o glucosa muy alta no son solo molestias: indican que variables internas se están saliendo de rango.

Ejemplo extra

La sed también es una señal homeostática. Si falta agua, el cuerpo activa respuestas para buscar líquido y conservarlo. No es una molestia cualquiera: es regulación para proteger el funcionamiento celular.
