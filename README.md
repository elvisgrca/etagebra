# EtaGebra - Academia de Ciencias

Sitio web oficial de **EtaGebra**, una academia en línea dedicada a la enseñanza de matemáticas, física, química, biología e inglés. El sitio proporciona información sobre la metodología de enseñanza, el equipo de profesores, testimonios y el catálogo completo de cursos ofrecidos.

## 🚀 Características del Sitio
- **Diseño Responsivo y Elegante**: Maquetado semántico en HTML5 con estilos personalizados y tipografías premium (`Space Grotesk` e `Inter`).
- **Catálogo de Cursos**: Detalle de cursos permanentes como Aritmética, Álgebra I y II, y Cálculo Diferencial.
- **Canal de Contacto Directo**: Enlaces automatizados a WhatsApp para inscripción y consultas directas con un mensaje personalizado.
- **Acceso Rápido**: Menú de navegación interactivo que funciona de manera dinámica tanto en ordenadores como en dispositivos móviles.

## 📁 Estructura del Proyecto
```text
etagebra_site/
├── assets/         # Logotipos e imágenes para cada curso
├── index.html      # Página de inicio principal (metodología, equipo, opiniones)
├── cursos.html     # Catálogo completo de asignaturas y detalles académicos
├── styles.css      # Hoja de estilos con variables CSS para el tema y componentes
└── script.js       # Control de comportamiento interactivo del sitio
```

## 🛠️ Tecnologías Utilizadas
- **HTML5**: Estructura semántica clara para accesibilidad y SEO.
- **CSS3 (Vanilla)**: Variables nativas de CSS, Grid, Flexbox y animaciones fluidas para una interfaz moderna.
- **JavaScript**: Lógica interactiva nativa (menú móvil, interactividad básica).

## 💻 Desarrollo Local
No requiere compiladores ni frameworks pesados. Puedes abrir directamente el archivo `index.html` en cualquier navegador web o usar una extensión como **Live Server** en tu editor de código para desarrollo local en vivo.
